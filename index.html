BEFORE YOU START

      
    
        
  • Your camera is processed entirely on-device — video is never uploaded, stored, or sent anywhere.
    • 
        
  • No cookies. No analytics. No server. This is a static page.
    • 
        
  • 4 keys saved to your browser's localStorage, nothing else: ac.consent (this consent), ac.mode (FULL/LIGHT mode), ac.bg (background choice), ac.osd (VHS overlay toggle).
    • 
        
  • The hand-tracking model file is fetched once from Google's MediaPipe CDN — a standard file request, no video or gesture data involved.
